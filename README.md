![SQLite](https://img.shields.io/badge/SQLite-Database-003B57?logo=sqlite)

## About

Project developed to study and understand the basics of **databases using Python**. The application explores in practice how to create, manipulate and update data in a local database, using a **Valorant**-themed dataset as its base.

---

## Technologies

- **Python** — Main language
- **SQLite** — Local database used in the application

---

## Features

- **Database Creation** — Structuring and creating tables
- **Data Insertion** — Registering new records
- **Data Update** — Modifying existing records
- **Data Query** — Reading and displaying stored information

---

## Project Structure

```
Banco-de-Dados---Python/
├── DATABASE.py         # Main script for database creation and manipulation
├── DatabaseUPDATE.py   # Data update script
└── Valorant.db         # SQLite database file
```

---

## How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/VryDeveloper/Banco-de-Dados---Python.git
   ```
2. Navigate to the project folder:
   ```bash
   cd Banco-de-Dados---Python
   ```
3. Run the main script:
   ```bash
   python DATABASE.py
   ```
4. To run the update script:
   ```bash
   python DatabaseUPDATE.py
   ```

> Make sure you have **Python 3** installed. No external libraries are required — the project only uses Python's built-in `sqlite3` module.

# Banco de Dados com Python

## Sobre o Projeto

Projeto desenvolvido com o objetivo de estudar e entender o básico de **banco de dados utilizando Python**. A aplicação explora na prática como criar, manipular e atualizar dados em um banco de dados local, utilizando como base um banco temático do jogo **Valorant**.

---

## 🛠️ Tecnologias Utilizadas

- **Python** — Linguagem principal do projeto
- **SQLite** — Banco de dados local utilizado na aplicação simples

---

## ✨ Funcionalidades

- 🗃️ **Criação do Banco** — Estruturação e criação das tabelas no banco de dados
- ➕ **Inserção de Dados** — Cadastro de registros no banco
- 🔄 **Atualização de Dados** — Modificação de registros existentes
- 🔍 **Consulta de Dados** — Leitura e exibição das informações armazenadas

---

## 📁 Estrutura do Projeto

```
Banco-de-Dados---Python/
├── DATABASE.py         # Script principal de criação e manipulação do banco
├── DatabaseUPDATE.py   # Script de atualização dos dados
└── Valorant.db         # Arquivo do banco de dados SQLite
```
