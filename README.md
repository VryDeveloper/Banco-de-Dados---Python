# Banco de Dados com Python

## Sobre o Projeto

Projeto desenvolvido com o objetivo de estudar e entender o básico de **banco de dados utilizando Python**. A aplicação explora na prática como criar, manipular e atualizar dados em um banco de dados local, utilizando como base um banco temático do jogo **Valorant**.

---

## 🛠️ Tecnologias Utilizadas

- **Python** — Linguagem principal do projeto
- **SQLite** — Banco de dados local utilizado na aplicação simples

---

## ✨ Funcionalidades

- 🗃️ **Criação do Banco** — Estruturação e criação das tabelas no banco de dados
- ➕ **Inserção de Dados** — Cadastro de registros no banco
- 🔄 **Atualização de Dados** — Modificação de registros existentes
- 🔍 **Consulta de Dados** — Leitura e exibição das informações armazenadas

---

## 📁 Estrutura do Projeto

```
Banco-de-Dados---Python/
├── DATABASE.py         # Script principal de criação e manipulação do banco
├── DatabaseUPDATE.py   # Script de atualização dos dados
└── Valorant.db         # Arquivo do banco de dados SQLite
```

---

## 🚀 Como Executar

1. Clone o repositório:
   ```bash
   git clone https://github.com/VryDeveloper/Banco-de-Dados---Python.git
   ```
2. Acesse a pasta do projeto:
   ```bash
   cd Banco-de-Dados---Python
   ```
3. Execute o script principal:
   ```bash
   python DATABASE.py
   ```
4. Para rodar as atualizações:
   ```bash
   python DatabaseUPDATE.py
   ```

> Certifique-se de ter o **Python 3** instalado. Nenhuma biblioteca externa é necessária — o projeto usa apenas o módulo `sqlite3` nativo do Python.
